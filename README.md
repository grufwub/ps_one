A simple terminal PS1 variable generator with git status support in Rust.

Simply add to your to your `.bashrc`/`.mkshrc`/`whatever`:
```
export PS1='$(ps_one)'
```

Pros:

- small

- fast

- sexy (subjective)

Cons:
